## 经典题
### 反转单向和双向链表
反转单向和双向链表，时间复杂度O(N)， 空间复杂度O(1)
```java
	public static ListNode reverseLinkedList( ListNode head){
		ListNode p = null;
		ListNode q = head;
		while(q != null){
			int save = q.next;
			q.next = p;
			p = q;
			q = save;
		}
		return q;
	}
```
